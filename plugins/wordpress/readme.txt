AI-powered accessibility scanning and fixes for WCAG 2.1 compliance.

== Description ==

AccessibleAI helps you achieve and maintain WCAG 2.1 compliance with automated scanning
and AI-powered fix suggestions.

= Features =
* Automated accessibility scanning
* WCAG 2.1 compliance checking
* AI-generated fix suggestions
* One-click fixes for common issues
* Detailed scan reports
* Accessibility score tracking
* WordPress dashboard widget
* Scheduled automatic scans

= Use Cases =
* Protect your business from ADA lawsuits
* Improve accessibility for all users
* Meet legal compliance requirements
* Enhance SEO and user experience

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accessible-ai` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to 'Accessibility' in your WordPress admin and connect your API key from [accessibleai.com](https://accessibleai.com)
4. Run your first scan!

== Frequently Asked Questions ==

= Do I need an account? =
Yes, you need an account at accessibleai.com to use this plugin. Sign up at https://accessibleai.com to get your API key.

= How often should I scan? =
We recommend at least weekly scans to catch new accessibility issues as you add content to your site.

= What does the accessibility score mean? =
The score is based on WCAG 2.1 guidelines. 100 means full compliance, while lower scores indicate issues that need to be addressed.

= Can I fix issues automatically? =
Many common issues can be fixed with one click. For complex issues, the plugin provides detailed fix suggestions that you or your developer can implement.

= Is this compliant with legal requirements? =
While using AccessibleAI helps you achieve WCAG 2.1 compliance, it does not guarantee legal protection. Consult with legal counsel for specific compliance requirements.

== Privacy Policy ==

AccessibleAI respects your privacy. Site content is scanned for accessibility issues and sent to our API for analysis. We do not store your full site content - only the accessibility issues and relevant context needed to generate fixes.
